# Attribution

## Summary

All code in this repository is **clean-room authored by NextCore contributors**. No source code is copied verbatim from any third-party project.

- Cross-IDE adapter system (10 IDEs: Claude Code, Antigravity, Cursor, Windsurf, Copilot, Continue.dev, Aider, Zed, JetBrains Junie, Void)
- Converter script (`adapters/antigravity/converter.cjs`) for automated Claude Code → target-IDE skill transformation
- Installer system with auto-detection, per-skill install, and 10-IDE support
- Skill catalog system (`skills/catalog.json` + `skills/catalog.html`)
- Agent performance skill patterns (context-budget, parallel-dispatch, skill-composition, router, response-format)
- NEXTCORE domain skills (facebook-dom, deploy-vps, nextcore-design, payment-integration, etc.)
